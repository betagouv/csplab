# Design system | ATS CSPLab

Ce book est la vitrine de référence de l'interface utilisateur de l'ATS (« Applicant Tracking Systems ») conçu par l'équipe **CSPLab** : il en présente les composants et documente les choix de conception.

---

## Architecture en 7 couches

Le book est organisé en plusieurs couches qui reflètent la structure du design system : des principes abstraits aux vues complexes, en passant par différentes échelles de composants.

| # | Couche | Rôle |
|---|--------|------|
| **00** | Design system | Méta, principes et organisation |
| **01** | Fondations | Tokens de design, couleurs, espacements, etc. |
| **02** | Atomes | Composants UI unitaires, sans logique métier |
| **03** | Molécules | Compositions UI génériques à base d'atomes |
| **04** | Composants métier | Atomes/molécules spécialisés pour le domaine ATS |
| **05** | Sections génériques | Blocs fonctionnels réutilisables dans toute webapp |
| **06** | Sections ATS | Assemblages métier riches |
| **07** | Vues ATS | Maquettes de pages complètes |
