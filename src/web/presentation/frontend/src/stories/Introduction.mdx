# Design system | ATS CSPLab

Ce book est la vitrine de référence de l'interface utilisateur de l'ATS (« Applicant Tracking Systems ») conçu par l'équipe **CSPLab** : il en présente les composants et documente les choix de conception.

---

## Organisation du book

Le book est organisé selon 4 niveaux de composition, des fondations jusqu'aux écrans.
Les niveaux intermédiaires utilisent un axe secondaire `Générique / ATS`.
Le détail des règles de classement est documenté dans `DDR-001`.

| # | Niveau | Rôle |
|---|--------|------|
| **00** | Design system | Introduction, DDR, principes et organisation |
| **01** | Fondations | Tokens, couleurs, espacements, états transverses |
| **02** | Éléments | Unités visuelles ou interactionnelles |
| **03** | Compositions | Assemblages fonctionnels, génériques ou ATS |
| **04** | Écrans | Vues complètes correspondant à un usage métier |
